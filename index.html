Cats versus Dogs (a game of world domination)

  • 2 sides (Cats: X's) and (Dogs: O's)
  • Game board is a 3 x 3 grid
  • Players alternate turns placing an avatar (cat or dog) in an available grid space
  • Victory conditions: 3 in a row (horizontal, vertical or diagonal)
  • Tie conditions - all spaces occupied but no victory condition met
